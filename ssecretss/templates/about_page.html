About

sSecretss!

sSecretss is the Simple Secret Sharing Server (or Service). It is shamelessly ripped off of Fugacio.us, which is awesome and free and open source and you should all go look at it, because sSecretss currently does nothing that Fugacio.us doesn't do more elegantly AFAIK.

That having been said, Fugacio.us does what it does in Ruby. I can't read Ruby. So I rewrote this in Python as a spiritual successor with (I think) more readable source code. It's built, currently, on Flask and SQLite. More robust options are in the works.

There'll be a GitHub link here when I've got the codebase prettied up enough that I can stand others to see it. :P
